# Healthive - Personalized Weight Management System

Healthive adalah aplikasi kesehatan berbasis web yang dirancang untuk membantu pengguna mencapai berat badan ideal melalui manajemen nutrisi yang dipersonalisasi. [cite: 338] [cite_start]Aplikasi ini menggunakan algoritma kesehatan standar (BMI, BMR, dan TDEE) untuk memberikan rekomendasi asupan kalori dan makronutrien yang disesuaikan dengan tujuan spesifik pengguna, baik untuk menaikkan maupun menurunkan berat badan. 

⚠️ **Status Proyek: Dalam Tahap Pengembangan** Proyek ini sedang dalam proses pengembangan aktif sebagai bagian dari perancangan Software Engineering. 

## Fitur Utama
- **Profil Kesehatan & Kalkulator BMI:** Penghitungan Indeks Massa Tubuh (BMI), Basal Metabolic Rate (BMR), dan Total Daily Energy Expenditure (TDEE) secara otomatis. 
- **Sistem Rekomendasi Nutrisi Personalisasi:** Memberikan target kalori harian berdasarkan tujuan (Naik/Turun BB) dengan pendekatan perubahan bertahap (*incremental change*). 
- **Pelacak Makanan Presisi:** Pencatatan asupan harian menggunakan basis data nutrisi per gram untuk akurasi tinggi. 
- **Rencana Makan (Meal Plan):** Rekomendasi menu makanan sehat dan akses ke perpustakaan nutrisi. 
- **Laporan Progres Visual:** Visualisasi tren berat badan dan pencapaian nutrisi harian/mingguan melalui grafik intuitif. 

## Teknologi yang Digunakan
- **Frontend:** HTML, JavaScript, CSS, TSX (Framework: Flutter/Dart diusulkan untuk mobile). 
- **Backend:** Python (Django/Flask). 
- **Database:** MySQL
- **Deployment:** Cloud Hosting (Vercel/AWS).

## Metodologi Pengembangan
Proyek ini dikembangkan menggunakan metode **Agile (Iteratif)** dengan kerangka kerja *System Development Life Cycle* (SDLC), yang memungkinkan penyesuaian cepat berdasarkan feedback pengguna.
